# Agentic RAG Study Helper — curated dependencies.
#
# This replaces the original global `pip freeze` dump, which was bloated
# (streamlit/jupyter/kubernetes/…), missing packages the code imports, and pinned
# tokenizers/huggingface_hub in a way that conflicted with sentence-transformers.
#
# Embeddings run on fastembed (ONNX) instead of sentence-transformers (PyTorch):
# same model (all-MiniLM-L6-v2, 384-dim), ~15 MB vs ~2.5 GB, no torch — much lighter
# for the free HF Spaces deploy. See strategy.md decision log.
#
# Versions are intentionally unpinned so pip resolves a mutually compatible set on
# Python 3.13. After a successful install, run `pip freeze > requirements.lock.txt`
# to capture exact versions for reproducible/prod builds.
